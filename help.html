Quick Start

  • Open the File menu for New, Open, Save, Save As, and Print.
  • Use the Formatting menu for fonts, headings, alignment, and lists.
  • Use Insert for tables and images.
  • Undo and redo up to 15 actions in the History section.

Find and Replace

  • Open Find with Ctrl/Cmd+F or the Find menu button.
  • Open Replace with Ctrl/Cmd+H.
  • Match case toggles exact case matching.

File Formats

  • TXT: plain text export and import.
  • Markdown: basic headings, lists, bold, italic, and inline code.
  • RTF: basic text and inline styles.
  • DOCX: export supports paragraphs, headings, lists; import is basic text, headings, lists.
  • PDF: export uses the browser print dialog.

Tables and Images

  • Insert tables from the Insert menu (rows and columns).
  • Insert images from your device; images are embedded in the document.
  • DOCX export represents images as [Image] placeholders.

Palettes and Theme

  • Use the Palette selector to theme the ribbon (5 variants).
  • Use the Theme button for light or dark mode.

Limitations

  • PDF import is not supported.
  • DOCX import is limited to basic text, headings, and lists.
  • Markdown support is intentionally lightweight.
  • Undo/redo tracks the last 15 content changes.
  • Paste may require clipboard permission depending on the browser.

Storage

  • Automatic snapshots are saved in localStorage (last 5).
  • No data is sent to a server; everything stays in the browser.
